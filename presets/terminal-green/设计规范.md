# Terminal Green · 终端绿 · 设计规范

## 气质
开发者、黑客美学

## 色彩

| 角色 | 色值 | 用途 |
|---|---|---|
| 背景 | #0d1117 | 卡片底色 |
| 背景叠加 | #161b22 | 卡片、区块底色 |
| 标题文字 | #39d353 | 主标题、强调文字 |
| 正文 | #8b949e | 可读内容 |
| 次要文字 | #606870 | 注释、页码 |
| 强调色 | #39d353 | 线条、标签、引用块 |
| 辅助色 | #58a6ff | 辅助点缀 |
| 外部背景 | #06090e | 卡片外围视口 |

## 字体

| 用途 | 字体 | 来源 |
|---|---|---|
| 显示/标题 | 'JetBrains Mono',monospace | Google Fonts |
| 正文 | 'JetBrains Mono',monospace | Google Fonts |

> 正文基准字号 36px。
> 同时包含中文和数字/拉丁字元的元素使用正文字体，避免字号不匹配。

## 标志元素

扫描线、闪烁光标、代码语法高亮

## 卡片结构

4 张示例卡片：
1. **封面** — 全幅图片 + 标题叠加
2. **左图右文** — 图片占左侧 44%
3. **上图下文** — 图片占上半 50%
4. **编辑式正文** — 首字下沉 + 引用块 + 内嵌图片

## 技术参数

- 画布：1080 × 1440 px (3:4)
- 边距：52px
- 正文：36px
- 导航：侧边按钮 + 右下角页码
- 输出：单 HTML 文件，内联 CSS/JS
