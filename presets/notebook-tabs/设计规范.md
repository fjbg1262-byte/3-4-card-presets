# Notebook Tabs · 笔记本标签 · 设计规范

## 气质
编辑感、精致、有质感

## 色彩

| 角色 | 色值 | 用途 |
|---|---|---|
| 背景 | #f8f6f1 | 卡片底色 |
| 背景叠加 | #f0ece4 | 卡片、区块底色 |
| 标题文字 | #1a1a1a | 主标题、强调文字 |
| 正文 | #555 | 可读内容 |
| 次要文字 | #999 | 注释、页码 |
| 强调色 | #98d4bb | 线条、标签、引用块 |
| 辅助色 | #c7b8ea | 辅助点缀 |
| 外部背景 | #2d2d2d | 卡片外围视口 |

## 字体

| 用途 | 字体 | 来源 |
|---|---|---|
| 显示/标题 | 'Bodoni Moda',serif | Google Fonts |
| 正文 | 'DM Sans',sans-serif | Google Fonts |

> 正文基准字号 40px。
> 同时包含中文和数字/拉丁字元的元素使用正文字体，避免字号不匹配。

## 标志元素

纸张阴影卡片、右侧彩色标签（竖排文字）、左侧打孔装饰

## 卡片结构

4 张示例卡片：
1. **封面** — 全幅图片 + 标题叠加
2. **左图右文** — 图片占左侧 44%
3. **上图下文** — 图片占上半 50%
4. **编辑式正文** — 首字下沉 + 引用块 + 内嵌图片

## 技术参数

- 画布：1080 × 1440 px (3:4)
- 边距：52px
- 正文：40px
- 导航：侧边按钮 + 右下角页码
- 输出：单 HTML 文件，内联 CSS/JS
